# 3D-Task-Tracker-with-Data-Structures-Vanilla-JS

An interactive Task Tracker Web Application built using Vanilla JavaScript, featuring real-time visualization of core Data Structures like Binary Search Trees (BST) and Graphs, along with a modern 3D-styled UI.

This project is designed to be both functional and educational, making data structures easy to understand through visual representation.
